# era悪魔でメイド：着床判定パッチ

## はじめに
現実において、人間の受精卵が着床に成功する確率はおよそ2割程度だそうです。  
これを口実に、era悪魔でメイドでの妊娠確率を引き下げるパッチを作ってみました。  
危険日に避妊なしで生セックスしたいけど妊娠はしてほしくないという身勝手な人や、  
射精量増加などの影響ですぐ妊娠してしまうのが不満な人向けです。  

## これはなに？
era悪魔でメイドに着床の概念を追加するパッチです。  
現状、era悪魔でメイドでは受精成功＝妊娠となっていますが、  
このパッチの影響下では受精成功後、着床判定にも成功しないと妊娠しません。

## 導入するには
### 対象バージョン
このパッチは、era悪魔でメイド ver0.054 (2019/02/05版)向けに作られています。
### インストール方法
ERBとCSVフォルダを上書きして下さい。  
うまくいかない場合は作者様が本体に取り入れてくれるように祈って待ちましょう。
### オプション設定
初期設定では着床判定がオフになっています。  
[555]コンフィグの[1]システムやイベント周りの設定から着床判定を有効にして下さい。

## 入れるとどうなるの？：直接的な影響
### 着床判定の追加
コンフィグで無効になっている間は入れる前と変化はありません。  
有効にすると、従来は受精後の妊娠確率が100%だったのが、デフォルトで40%まで下がります。  
また、受精から着床まで何日かかかるようになります。  

### 素質と着床
素質：妊娠しづらいを持っていると着床確率は通常より下がります。
かなり妊娠しづらくなります。

### 素質：妊娠しやすい追加
着床しやすくなる素質：妊娠しやすいが追加されます。

### アフターピル
避妊に失敗した場合などに、アフターピルを飲んでもらうよう、お願いできるようになります。  
アフターピルは受精は防げませんが、受精卵の着床を防止する効果があります。  
アフターピルは、使用から数日の間有効です。  

### 避妊の護符の貫通
注：この要素はコンフィグでOFFにできます。デフォルトはOFFです。  
避妊の護符が弱体化され、「効果が＊ほぼ確実に＊着床を防ぐ」というものになります。  
避妊の護符を持っていようと受精する上に、稀にですが着床・妊娠してしまいます。  
つまり、信仰の力で避妊している神の娘を、わずかな確率とはいえ合意無しで孕ませられるようになります。  
ロマンですね。本人がそのことを知っている場合、知らない場合どちらもそれぞれおいしいシチュです。  
ただし、狙って妊娠させるのは相当難しいぐらいの低確率(着床率 3%)です。

## 間接的な影響
### 「妊娠しやすくする」効果の価値UP
狙って妊娠させるのが難しくなるので、妊娠しやすくする効果に価値が出来ます。  
神社での祈願や、聖職者系の「お願い」で着床確率を上げられても面白そうです。  
例えば、マノンさんがぽこぽこ産んでるのに、一人も出来ず寂しそうなフェリシエルさんの危険日に  
白蛇神社で子宝祈願し、夜には本気の子づくりセックスするとか、  
マリーさんに着床確率アップのお願いをして「どなたとですか？」という問いにキミとだけどと答えてみるとか、  
パラチネさんに避妊の護符解除をお願い後、即着床確率アップしてもらってあえて放置して寝るとか、  
妄想が広がります。

### 素質への影響
本パッチ無しでは、妊娠しづらいキャラであっても、妊娠させることは案外容易です。  
着床判定パッチにより、素質：妊娠しづらいをもっているキャラの着床確率は、  
持っていないキャラの6割まで低下し、かなり妊娠しづらくなります。  
妊娠しづらいの強度によっては着床確率が数%にまで低下します。  
これによって、極端に妊娠しづらいキャラ（極めて長寿だったり、霊的に高位の存在とか？）を  
より表現しやすくなります。  
また、素質：妊娠しやすいでは着床確率が上がります。  
素質強度によっては100%を越えるので、(普通のキャラが妊娠しづらくなるのもあって)避妊しないとすぐ孕んじゃう、  
みたいなキャラも表現しやすくなります。

### 避妊の重要性が下がらない？
着床確率が100%ではなくなり、避妊なしでも妊娠しないことは多くなります。  
ですが相変わらず、膣内射精一回で妊娠してしまうことはあり得るので、
結局、妊娠させたくなければ避妊が必要なのは変わらないでしょう。
