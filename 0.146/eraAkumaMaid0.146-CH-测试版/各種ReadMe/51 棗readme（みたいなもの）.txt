＊なにこれ
長身巨乳巨尻魔法使い幼馴染を悪魔メイドに召喚するあれやそれやです。
転校生要素も含んでいる為、１週間たたないと勧誘できません。

製作者が口上初心者なので不具合が多々発生する場合がございます。
その際は報告していただけると幸いです。

＊導入方法
ERBをぽいっと入れて上書きしてください。

＊参考口上とか
フェリシエル口上、ノノ口上、みなみ口上を参考かつ分岐などを流用させていただいております。
この場をお借りして感謝の意を表明いたします。ありがとうございました。

＊ライセンス
悪魔メイドに限り改変・再配布はOKです。


＊やること
・乳関連をすべて埋める（最重要）
・初回調教以外を3パターンぐらい作る（それなりに重要）
・部員同士のかけあい（それなりに重要）
