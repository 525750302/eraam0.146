作者様へ：本口上で使われている特殊ギミックについて

一部で本来想定されていないと思われるギミックを仕込んでいます。
もし問題になりそうな点にお気付きでしたらぜひご一報ください。

・口上の初回起動時にモード選択＆ニ人称変更の選択肢がある
肩書きを口上内で書き換えることで実現しています。
また、魂の取引に追加されたニ人称変更機能を呼び出す選択肢を追加してあります。

・DITEMTYPEによる一部コマンドの禁止と合意：本番の使用
序盤の挿入コマンド禁止のためにDITEMTYPEを使用しています。
また、合意に「本番/」を追加することで「挿入の許可を貰った」というフラグ代わりにしています。

・素股で特定条件を満たすとコマンドが正常位に変化し、正常位としてフラグやソースの処理が行われる
BEFORE_COM内でSELECTCOMを60に書き換えることで実現しています。

・フェリシエルがアドバイザーの場合、少し喋る
キャラ描写に違和感があるようなら修正しちゃってください。
