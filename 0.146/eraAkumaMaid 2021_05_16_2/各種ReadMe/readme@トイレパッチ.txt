;===========================================================
;  era悪魔でメイド用 トイレハプニングパッチ
;===========================================================

1. 概要
　era悪魔でメイド にトイレでばったりなイベントを追加するパッチです

2. 対応バージョン
　eraAkumaMaid 2019_05_05 にて作成しています

3. 導入方法
　ERBフォルダをバリアントフォルダの同名フォルダに上書きしてください
（ERBファイルを上書きすることになるので、確認を求められたときは
　そのまま実行してください）

4. 簡単な仕様
　・以下の条件を満たすとき、確率でイベントが発生します
　　・同棲相手が複数居る
　　・場所が自宅である
　　・同棲しているヒロインの尿意が一定以上
　・イベント中、特定の条件を満たすとイベントの内容が派生します
