2021/04/30　ver0.05
これを読んでいると言うことは、あなたはちゃんとReadMeを読む人ですね。
まぁファイルの名前の通り「読んでね」と言う名前なので当たり前と言えば当たり前かもしれませんが。
仮にこれを読まなくてもそれを批判する意は全くありません。それは個人の自由ですから。
と、長々と前語りなぞ書いても仕方ないので、簡潔に伝えたいことを。

「一部イベントの書き直しと、それの為の描写を増やしました」
「そして、とあるイベントを全文書き直しました」

この千紗と言う少女が何を思って何を考えてるか、一番大事な部分が伝わりにくいと言う言葉を受けました。
言い訳をするのであれば、元々思考実験で出来上がったのでキャラとしての下地が薄かったのが原因でした。
そうやって、どう言うキャラなのかを詰めていくと、とあるイベントが全文書き直さないと意味が無いと気が付きました。
全く持って間抜けな話です。まぁ、大半のプレイヤーにとってはどうでも良い事かもしれませんが。
エロをしに来ているのに長々とだらだらとエロとは関係のない話を展開されて。
そう言うのを書きたいのであればSSでも書いたら？と言われたらその通りです。
肝心のエロ口上もそんなに多くないという始末。
恥じ入る限りです。

長々と書きましたが結論は「実質的にリビルドになりました」でした。
これを本当なら4月4日に出したかった。
等と後悔を言っても仕方なし。後に悔いるのが後悔ですから。

取り合えずの自分の感覚では良くなったと思っています。
しかし、そう思わない人が居るのは確かだと思います。
でもそれは、ある種仕方のない事だと思います。
そんな事をしている暇があれば空白の部分を書けと言われるのも普通だと思います。

……全くもって私は、言い訳がましいですね。

それでも、千紗と言う少女に触れて貰えるのでしたら幸いです。

一年経って今更ながらに千紗と言う少女と「私」を語る。
ここから先は2021/4/30までに書き直した部分のネタバレとなるので注意。
作者の内面が、がっつり出るので知りたくない人は見なかったことにして閉じるを押すべし。
それでも読むというならば、思考を適当に書き綴るだけなので恐らく読みにくいと思われる。
文才の無い証拠だ。クソッたれが。

さて、千紗と言う少女は最初は思考実験で生まれたキャラクターだった。
「殺す事しか知らない存在が、殺しても死なない存在と出会うとどうなるだろう？」
一風変わってもいない、ごく在り来たりな思考だ。
それを考えた瞬間に「花の意味、言葉を選ぶなら」のシーンが浮かんだ。
殺すことを続けるのか、やめて良いのかの問答のシーンが。

それをdiscordで話したら思った以上に反応を貰えて書き始めたのが始まりだった。

口上を書くのが初めてでわからないことが色々とあったが、そこは先達に色々と聞いて書いた。
それでもこの時、私が千紗と言う少女をちゃんと捉えていることが出来ていたかと問われると、
結果から見ると全然そんな事なかった。

生い立ちに赤ずきんをモチーフにしたが、それをキャラとしては全く生かせていなかった。
当時は、オオカミに食べられそうだった赤ずきんが間違ってオオカミを殺して、オオカミになってしまった。
と言うモノだった。これ自体は間違ってないと思う。何せ先人たちが散々と使い古したネタであろうから。

でも、それを生かせてなった。
その結果、個別イベントはあるけれどキャラ単品で見ればそんなに活いてないと言うことになった。
つまり、個別イベントでは舞台装置、日常では発展性のない微妙さ。
私が、かわいいと思える反応は書ける。書けるが、当時のキャラ性だとそれ以外が思いつかなかったのだ。
それが、ダイレクトに反映されたのか更新するたびに個別イベントだけが増えていった。
しかしそれに気が付いていたのかと問われるとそんなことはなかった。

漠然と何か思いつかないなとかは思っていた。そう、思っていただけなのだ。
そんな折に、ちょっとキャラ性が薄いよと言ってくれた人が居た。それが3月末。
正しくは全体的にちぐはぐと言うのが良いのだろうか？兎も角、完成度が低いとハッキリと言ってくれた。
それから千紗と言う少女の事を考えた。
取り合えず今まで書いたモノからあなたに出会う前の少女が内在的に抱えている方向性を洗い出した。

性格
素直・疑いを知らない・短絡的・刹那的・幼い・好奇心旺盛・落ち着きがない
気性
拮抗した選択肢に弱い・怖がり・直感で合理性を選ぶ・頭の回転は良し・強欲・保守的
その他
孤独・実践派・何故を考えない

今まで書いたモノから出てきたのはこれだった。
親から虐待され、殺してしまった少女は暴力を愛だと思った、思いこんだ。これだとこれまで通りだ。
では、何故そうなったのか。恐らくこの少女は両親が好きだったのだろう。だからそれを捨てたくなかった。
だから、これがわたしが貰った愛だと人を殺した。そこまで少女自身が考える事を出来ると気が付いた時、私がふと思ったのは

この少女は「ボク」と言うペルソナを演じているかもしれない、だった。

そう思うとすとんと来た。
メタ的な理由で元々「ボク」と言う一人称を使ったのはプレイヤーであるあなたに幼い印象を与えるためだった。
しかし、だ。少なくとも数年は親元で育てられた少女が「ボク」と言う一人称を使うのは特別な理由がない限りあり得ないだろう。
であれば、それを使うとしたらどんな理由だ？と考えた。そう、演じるだ。
あどけない少女を演じて、相手の油断を誘って人を殺す。そう言う事をしているのだと。

しかし、この少女は人を殺す意味を知っているはずだ。
こうなるとコンセプトと矛盾が発生する。

結果、「ボク」と「わたし」と言う表層と深層を分ける事にした。
普段、と言うよりはあなたと出会ってからそこに居る千紗と言う少女は「ボク」である。
だけれども、それが千紗と言う少女の全てかと言われるとそんなことは無かった。
冷静に考えている部分がある。でもそれを明かす方法があなたでは無理だった。
いや、出来るけれども、それはプレイヤーから乖離してしまう。
散々プレイヤーを無視した動きをさせておいて何を言うと言われると……何も言えないです。
