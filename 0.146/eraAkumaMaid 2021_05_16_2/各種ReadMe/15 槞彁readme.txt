槞彁について
　魔界からやって来て間もない悪魔の少女です。
　種族悪魔ですが職業悪魔ではありません。こちらの世界に不慣れなので毎日大変そうです。
　こちらの言葉を話す事が出来ないようですが機械翻訳で必死に意思の疎通を取ろうとしています。
　しかし、機械翻訳の精度は良く無いようなので意味不明な文章になる事もあるでしょう
　彼女に付き合って、伝えたい言葉を推測してあげてください
